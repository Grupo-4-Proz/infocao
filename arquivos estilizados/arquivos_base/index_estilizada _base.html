Caso queira utilizar a janela lateral esse é o link de referência

Monte sua página dentro desse article. Só deverá aparecer a sua página aqui. Exclua essa frase e o link acima das articles que não forem suas para funcionar corretamente. Traga o seu conteúdo da index na pasta arquivos desatualizados para cá.

As bordas coloridas que você vê são apenas guias para facilitar a nossa percepção sobre o tamanho dos elementos. Caso queira excluir é só ir no elemento corresponder e apagar o "border" e seus atributos

Altere o id do article id="slide_tema" para o id correspondente ao seu tema, conforme o exemplo a seguir: slide_origem_caes slide_coleta_dados slide_adocao slide_ongs slide_mural. Isso será importante no momento que formos unir todo o conteúdo numa página. Lembrem-se de colocar classes e ids que sejam claros e façam referência a sua página e seus elementos, pois futuramente juntaremos todos os arquivos e se utilizarmos nomes genéricos, pode dar conflito. Exemplo: Não chamar uma imagem de imagem_1, etc.
